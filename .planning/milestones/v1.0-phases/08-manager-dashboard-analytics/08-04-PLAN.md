   - Custom SVG dot matrix visualization (per locked decision: dots with size + color encoding, NOT traditional colored grid)
   - Rows: team members (or "Member 1/2/3" when anonymized), Columns: question categories
   - Each cell: a circle/dot where:
     - SIZE maps to sample count (more data = larger dot, min 6px radius, max 16px)
     - COLOR maps to score value using green-yellow-red gradient:
       - Score >= 4.0: green (hsl(142, 76%, 36%))
       - Score 3.0-3.9: amber (hsl(38, 92%, 50%))
       - Score < 3.0: red (hsl(0, 72%, 51%))
     - Missing data: empty cell (no dot)
   - Row labels on left, column labels on top (rotated 45deg for space)
   - Hover tooltip: shows exact score, sample count, member name, category
   - Cells with < 3 data points: show small hollow circle (outline only) to indicate insufficient data
   - Responsive: horizontal scroll wrapper on mobile
   - Props: `{ data: Array<{ userId: string; userName: string; category: string; score: number; sampleCount: number }>; categories: string[] }`
