    ```
    {
      chatMessage: z.string().min(1),  // AI's explanation to the user
      templateJson: z.object({         // Full replacement template (internal protocol)
        schemaVersion: z.literal(1),
        language: z.string(),
        name: z.string(),

        sections: z.array(z.object({

            answerConfig: z.record(z.unknown()),
            isRequired: z.boolean(),
            sortOrder: z.number(),
            scoreWeight: z.number(),
            conditionalOnQuestionSortOrder: z.number().nullable(),
            conditionalOperator: z.enum(["eq","neq","lt","gt","lte","gte"]).nullable(),
            conditionalValue: z.string().nullable(),
          }))
        })).min(1),
      }).nullable(),
    }
    ```

    Export: `templateChatResponseSchema`, `type ChatTurnResponse = z.infer<typeof templateChatResponseSchema>`.
