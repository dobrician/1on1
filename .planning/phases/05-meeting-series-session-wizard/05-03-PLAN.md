    **Context panel (`context-panel.tsx`):**
    "use client" component. Always-visible right sidebar within the wizard layout.

    Props:
    ```typescript
    interface ContextPanelProps {
      currentStep: number; // 0 = recap, 1..N = categories
      currentCategory: string | null; // null on recap/summary
      previousSessions: PreviousSession[];
      openActionItems: OpenActionItem[];
      sessionScores: number[]; // last 6 scores for sparkline
      onQuestionHistoryOpen: (questionId: string) => void;
    }
    ```

    Behavior by step:
    - **Recap step (step 0):** Show general context:
      - Score trend sparkline (last 6 sessions) with label "Score Trend"
      - Open action items from all categories, grouped by session number
      - Overall summary stats: total sessions, average score
    - **Category steps (step 1..N):** Show category-specific context:
      - Previous session's notes for THIS category (from sharedNotes JSONB[currentCategory])
      - Previous session's answers for questions in THIS category (collapsed, expandable)
      - Action items tagged with THIS category
      - Small button on each question: "View history" triggers onQuestionHistoryOpen(questionId)

    Layout:
    - Fixed width sidebar: `w-[320px] shrink-0 border-l bg-muted/10 overflow-y-auto`
    - Collapsible on mobile: hidden by default on screens < lg, toggle button to show as overlay
    - Sections with clear headings: "Previous Notes", "Previous Answers", "Open Action Items", "Score Trend"
    - Each section is a collapsible Card (use shadcn Collapsible or custom toggle)
    - Empty states for each section when no historical data exists ("No previous notes for this category")

    **Question history dialog (`question-history-dialog.tsx`):**
    "use client" component. Opens when user clicks the "View history" button/icon on a question.

    Props:
    ```typescript
    interface QuestionHistoryDialogProps {
      open: boolean;
      onOpenChange: (open: boolean) => void;
      questionId: string;
      questionText: string;
      answerType: string;
      previousSessions: PreviousSession[];
    }
    ```

    Content:
    - Dialog (shadcn Dialog) with title = question text
    - Shows a timeline of previous answers for this specific question across last 3-6 sessions
    - Each entry: session date, session number, the answer value (formatted per answerType)
    - For numeric types: show a mini sparkline of just this question's values
    - For text types: show the full text (collapsible if long)
    - For yes/no: show colored badges (green Yes, red No)
    - Empty state if this question has never been answered before

    **Score sparkline (`score-sparkline.tsx`):**
    "use client" component using Recharts.

    ```typescript
    interface ScoreSparklineProps {
      data: number[];
      color?: string;
      height?: number;
    }
    ```

    Implementation:
    - Use Recharts LineChart with ResponsiveContainer
    - Hide all axes (no XAxis, no YAxis, no CartesianGrid, no Tooltip by default)
    - Single Line with type="monotone", strokeWidth=2, dot=false
    - Color defaults to `hsl(var(--primary))`
    - Optional: show a subtle dot on the most recent data point
    - Height defaults to 40px
    - If data has < 2 points, show a dash ("-") instead of sparkline
