   ```typescript
   async function switchLanguage(lang: 'en' | 'ro') {
     if (lang === currentLang) return;
     // 1. Save to DB + set cookie via API
     await fetch('/api/user/language', {
       method: 'PATCH',
       headers: { 'Content-Type': 'application/json' },
       body: JSON.stringify({ language: lang }),
     });
     // 2. Update the session JWT so subsequent requests use new locale
     await update(); // from useSession -- triggers JWT refresh
     // 3. Reload page to re-render with new translations
     window.location.reload();
   }
   ```
   - Import `update` from `useSession()` -- destructure as `const { data: session, update } = useSession();`
   - The `update()` call triggers the JWT callback with `trigger: "update"`, which Plan 01 set up to re-fetch the user's language from DB
   - The page reload is needed because next-intl's server-side message loading needs to re-run with the new cookie

3. Use proper Romanian display name: "Romana" with diacritics. The language names should be in their own language (English = "English", Romanian = "Romana" with proper chars).
