5. **Replace manual relative time strings** ("X days ago", "today", "yesterday") with `format.relativeTime(new Date(dateValue))` which outputs locale-aware relative time.

6. **Replace any remaining hardcoded English strings** (like "No sessions", "Overdue") with `t()` calls if not already using translations. These components mostly already use `useTranslations` -- check for any remaining hardcoded toast messages or labels.

Specific files and known issues:

- `upcoming-sessions.tsx`: Has `toLocaleDateString("en-US")` -- replace with `format.dateTime()`

- `action-items-page.tsx`: Has `toLocaleDateString("en-US")` and already uses `useFormatter` -- ensure consistent usage, remove any remaining "en-US" hardcoded calls
